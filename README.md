## Development Setup

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:8080
$ yarn run dev

# build for production with minification
$ yarn run build

# lint code
$ yarn run lint

# run unit tests using mocha + chai
$ yarn run unit

# run e2e tests using wdio
$ yarn run e2e

# run full test suite
$ yarn run test
```
